Nuage de mots

Description:
    Ce projet est une application web permettant de générer des nuages de mots à partir de textes saisis ou importés.
    Les mots sont visualisés avec une taille proportionnelle à leur nombre d'occurence dans le texte.

Fonctionnalités:
    Saisie directe de texte : Entre du texte dans une zone dédiée.
    Importation de fichier texte : Charge un fichier .txt pour analyser son contenu.
    Génération de nuage de mots : Crée un nuage de mots interactif à partir des données.
    Téléchargement du nuage : Télécharge le nuage de mots au format PNG.

Technologies utilisées:
    HTML, CSS, JavaScript (avec WordCloud2.js pour générer le nuage de mot).
    Python (Flask) pour traiter le texte et générer les données de fréquence des mots.

Utilisation:
    Lancer projet.py
    Ouvrir index.html dans un navigateur.
    Saisir du texte ou importe un fichier ".txt".
    Cliquez sur Analyser pour générer un nuage de mots.
    Cliquez sur Télécharger pour sauvegarder l'image.
